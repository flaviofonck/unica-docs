    1.2 Tabla central — DIM_CONFIGURACION_SERVICIOS¶

    Esta tabla es la fuente de verdad del sistema. Es referenciada por todos los módulos transaccionales. Contiene 50 campos que cubren desde datos comerciales hasta estados de baja y cálculos de terminación anticipada.

    Módulos que la leen o actualizan:

    Módulo Operación Campo(s) afectados
    HU-03 Migraciones UPDATE proveedor, mrc, medio, bw al registrar fecha_migracion_realizada
    HU-04 Bajas UPDATE estado_baja, fecha_baja, fecha_tope_pago_baja, pago_proyectado_preaviso
    HU-04 Reconexión UPDATE estado_configuracion → ACTIVO, fecha_alta
    HU-05 Contratos UPDATE condicion_pago, condicion_facturacion (trigger desde dim_contratos)
    HU-06 Prorrateo READ condicion_facturacion, mrc, moneda_mrc, novedad, fecha_novedad
    HU-07 Indisponibilidades READ ds, id_servicio, fecha_baja, proveedor
    HU-08 Prefacturación READ id_servicio, id_configuracion_servicio (vía CUS = DS)
