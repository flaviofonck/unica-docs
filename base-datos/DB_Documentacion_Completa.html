    PROCESOS ÚNICA — Documentación Completa de Base de Datos¶

    Instrucciones de uso para Claude¶

    Este documento es la referencia autoritativa del modelo de datos del sistema PROCESOS ÚNICA (QCS/Telefónica). Úsalo para:

    • Responder preguntas sobre estructura de tablas, campos y relaciones.

    • Validar FKs y cardinalidades antes de escribir consultas SQL o código.

    • Generar migraciones, scripts DDL o documentación técnica adicional.

    • Detectar inconsistencias entre HUs y el modelo de datos.

    Versión: Actualizada a HU-01 v3.0 (27/02/2026) y HU-02 v3.0 (04/03/2026).
    Separador CSV: punto y coma (;).
    Motor de base de datos objetivo: PostgreSQL.
    Convenciones: snake_case para todos los identificadores. PKs siempre BIGSERIAL o VARCHAR según se indica. FKs referenciales pueden ser NULL salvo indicación NN.
